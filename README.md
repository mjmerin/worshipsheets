# WorshipSheets

Worship sheets is intended to create worship lyric sheets for bible studies by using a string of URLs from popular Christian music websites such as pnwchords, essentialworship, worshipchords, worshiptogether and other similar websites. 

## Websites supported:

[WorshipChords](https://worshipchords.com/)

## Usage

```
python3 worshipsheets.py <URL>
```
